可以使用 内联 CSS 样式、内嵌 CSS 样式、或者外部 CSS 样式文件

fill 设置对象内部的颜色， stroke 绘制对象的线条颜色

颜色可以是使用 颜色名字 red, rgb\rgba,十六进制等等合法的颜色值, fill-opacity & stroke-opacity 可以用来设置 对应颜色的不透明度

stroke-linecap 用来控制描边的方式，他控制边框终点的形状

butt 用直边结束线段，线段边界90度垂直于描边的方向、贯穿他的终点（粗图线条） ； square（平方线条）；round 边框是圆角的，圆角的半径有 stroke-width 控制

stroke-linejoin 用来控制描边的方式，他控制边框终点的形状

miter 默认值，直角连接； round 圆角连接 bevel 斜面连接

stroke-dasharray="areaLen1 areaLen2 areaLen3" 可以将虚线类型应用在锚边上
