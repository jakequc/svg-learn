绘制平滑曲线的命令有三个： 其中两个是用来绘制贝塞尔曲线的，另外一个是用来绘制弧形或者说圆形的一部分。 在 path 元素里，只存在两种贝塞尔曲线： 三次贝塞尔曲线 C 和 二次贝塞尔曲线 Q

三次贝塞尔曲线 path 中的 C

三次贝塞尔曲线 C， 三次贝塞尔曲线需要定义一个点和两个控制点，所以用 C 命令创建三次贝塞尔曲线，需要设置三组坐标参数

C x1 y1, x2, y2, x y or c dx1 dy1, dx2 dy2, dx, dy

这里的 (x,y) 表示的是曲线的终点，另外两个坐标是控制点，（x1,y1）的起点是控制点，（x2,y2） 是终点的控制点； 控制描述的是曲线起始点的斜率，曲线上各个点的斜率，是从起始斜率到终点斜率的渐变过程

三次贝塞尔曲线命令 - S

二次贝塞尔曲线命令 - Q


弧形命令 - A

弧形命令A时创建一个 svg 的曲线命令，弧形可以视为 圆形 或者 椭圆形的一部分。假设，一直椭圆形的半长轴和短半轴，并且一直两个点（在椭圆上），根据半径和两个点，根据亮点可以画出两种弧线，即仅仅根据半径和亮点，可以画出两种弧线，为了保证创建弧形唯一，A 命令有较多的参数： A/a rx ry x-axis-rotation large-arc-flag sweep-falg x y rx 是 x 轴的半径，ry 是 y 轴的半径 ；x-axis-rotation 表示 x 轴的旋转情况; large-arc-flag(角度大小)决定弧线是大于还是小于 180 度，0 表示 小角弧，1 表示 大角弧； sweep-flag 表示弧线的方向， 0表示从起点开始到终点沿逆时针画弧，1 表示从 起点到终点沿顺时针画弧

large-arc-flag(角度是否大于180) sweep-falg（从起点到终点是否为顺时针）
